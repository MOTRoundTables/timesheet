# Product Overview

## Timesheet Automation Project

A comprehensive GUI application that automates the process of filling out online timesheets by integrating Excel data management with Google Calendar and web automation.

### Core Purpose
- Reads timesheet data from Excel files with Hebrew column headers
- Integrates with Google Calendar to import events automatically
- Uses Selenium WebDriver to automate data entry into web-based timesheet systems
- Provides data validation and conflict resolution tools

### Key Features
- **Automated Login & Data Entry**: Logs into timesheet systems and fills forms automatically
- **Google Calendar Integration**: Imports calendar events with intelligent conflict resolution
- **Excel Data Management**: Validates overlapping entries, manual entry forms, backup creation
- **GUI Workflow**: Step-by-step guided process (Get Events → Review/Edit → Run Automation)
- **Data Integrity**: Automatic backups, overlap detection, and interactive resolution dialogs

### Target Users
Employees who need to fill out web-based timesheets regularly and want to automate the process using their existing calendar data and Excel workflows.
